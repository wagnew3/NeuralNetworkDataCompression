PAQ8o6 Linux binaries. Command-line.
It's only compiled binaries from PAQ sources. 
There are three files: 
paq8o6_64 - AMD64 version, 
paq8o6_sse - SSE2 version, optimized compilation
paq8o6 - Generic MMX version.

Also includes benchmark results.
